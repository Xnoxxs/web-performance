# Web Performance

**Name:** Wayfair

**URL:** https://www.wayfair.com/

## About the Website

Wayfair is one of the world's largest online furniture and home goods retailers. The website offers millions of products across categories such as furniture, décor, lighting, kitchenware, outdoor living, and home improvement. It provides a modern e-commerce experience with product search, filtering, recommendations, shopping cart functionality, user accounts, and online checkout.

## Why Audit Wayfair?

Wayfair is a strong candidate for a web performance audit because it contains a wide variety of content and functionality, including:

- Large image galleries
- Dynamic product listings
- Search and filtering
- User authentication
- Shopping cart and checkout
- Personalized recommendations
- API-driven content
- Interactive user interface

These features make it possible to analyze different aspects of website performance and identify meaningful optimization opportunities.
